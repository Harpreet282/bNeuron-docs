# License Keys (On-Prem)

## 🔑 Key Listing

Admin will see active license keys with:

- Expiry Date  
- Assigned Tenant  

## 📥 Upload New Key

- **License File:** Admin will choose the key file from local drive.  
- **Activate:** Admin will click **“Upload & Activate.”**
